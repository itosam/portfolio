# Sam Ito Portfolio

Personal portfolio site for Sam Ito, focused on web product work, eCommerce content, SEO, website QA, creative assets, and full stack development projects.

## Features

- Responsive portfolio homepage with animated intro text
- Project landing page for code projects and creative work
- Creative case study/gallery page for Asian World of Martial Arts assets
- Resume page with real PDF loading state
- Contact form powered by Getform with honeypot spam protection
- Accessible focus states, image alt text, and keyboard-friendly links
- SEO metadata, Open Graph/Twitter tags, route-based page titles, and JSON-LD structured data
- Shared highlight/button interaction styles across navigation, links, and buttons

## Built With

- React
- React Router
- Tailwind CSS
- Framer Motion
- React Icons
- Getform

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm start
```

Create a production build:

```bash
npm run build
```

## Project Structure

- `src/components` - React page and UI components
- `src/data` - Project data used by the portfolio cards
- `src/pdf` - Resume PDF
- `src/styles` - Shared Tailwind class constants
- `public/assets` - Site images, project previews, and creative assets

## Deployment

The app is configured for GitHub Pages through the `homepage` field in `package.json`.
